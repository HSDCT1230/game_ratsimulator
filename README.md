# 鼠鼠模拟器 2.0

一个纯静态 HTML 鼠鼠养成小游戏。当前线上入口保持和 1.0 一致，为根目录的 `index.html`。

## 本地运行

双击 `run-local.cmd`，或直接用浏览器打开 `index.html`。

## 目录结构

```text
index.html          2.0 游戏主文件
bgm/                2.0 背景音乐
sound effect/       2.0 音效资源
archive/v1.0/       1.0 版本归档，保留旧版同名结构
docs/               规则备忘录、部署说明、音效来源
tools/              开发辅助工具，例如音效试听页
run-local.cmd       本地快速打开游戏
```

## GitHub Pages

GitHub Pages 直接使用仓库根目录发布。根目录中的 `index.html`、`bgm/`、`sound effect/` 是 2.0 正式运行文件；`archive/`、`docs/`、`tools/` 仅用于归档和维护。
